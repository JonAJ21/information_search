### Инструкция по запуску

1. Запуск контейнеров: docker-compose up --build
2. SwaggerUI: http://localhost:8000/api/docs
3. Для старта обкачки:

   3.1. Убедиться, что в SearchRobot/app/config/config.yaml прописаны sitemaps, которые необходимо обкачать.

   3.2. Запустить поикового робота с помощью API.
5. Для работы с поисковой системой:

   4.1. Убедиться, что в MongoDB содержаться данные о страницах.

   4.2. Если используем собранный корпус документов, то импортируем его с помощью mongo-express
7. Запуск приложения может занимать время, т.к. при его инициализации индекс собирается заново.
